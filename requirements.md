# Требования к приложению TaskManager

## Функциональные
- Добавление задачи (ID, название, описание, статус)
- Просмотр всех задач
- Изменение статуса
- Редактирование и удаление
- Автосохранение в JSON

## Нефункциональные
- Данные хранятся в `tasks.json`
- Не падает при ошибочном вводе
- Простая структура без сторонних зависимостей

## User Stories
1. Как пользователь, я хочу добавить задачу «Сдать проект», чтобы не забыть дедлайн.  
2. Как пользователь, я хочу видеть статус задач, чтобы понимать прогресс.  
3. Как пользователь, я хочу редактировать и удалять задачи, чтобы держать список актуальным.
