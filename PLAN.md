# План проекта TaskManager

## Цель
Создать консольное приложение для управления задачами с сохранением данных в JSON.

## Пользователи
Студенты, разработчики, все кто хочет просто вести список дел в терминале.

## Архитектура
- `data.py` — работа с файлом tasks.json  
- `logic.py` — бизнес-логика  
- `ui.py` — консольный интерфейс  

## Технологии
Python 3.9+, Git, JSON, UML (PlantUML/текстовая схема)
